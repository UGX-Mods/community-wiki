UGXLIN supports two major  Network Settings :

Max download speed

Set the max download speed (for file downloads only) which the application is allowed to use.

Per default, it will try to use the max available download speed possible. (= unlimited speed)

 

HTTP Proxy

Set the proxy settings which will be used for the application.
Per default, UGXLIN will use the system defined proxy.

 

You can also choose to use no proxy for the application  only , or
choose custom proxy settings for the application only :

 

You need to enter at least the proxy address (also known as host) and you might need to change the default port.

The Username and  Password fields are optional .

 

Note

currently the settings are not persistent, which means once you restart the UGXL Installer, you have to set your settings again .
