The UGXL Installer is written from scratch.

You can help us to improve the installer by verifying if everything worked correctly.

This page is especially helpful, if you encounter issues and you would like to report us more information why and where the issue happens.

Start Debugging

You need to start the application in DEBUG mode:

  1. Either make a shortcut of the executable...

  2. or create a *.bat file in the same folder as the UGXL Installer
    And give it this content:

    Debug.bat 
    
"UGXL Installer.exe" -d

If you sccessfully enabled the debug mode, an additional window should open:



Logfile

Note

While in beta, the logfile writing is forced!

The logfile path is:

%TEMP%\UGXLIN

and is called

ugxlin.log
