With the UGXL Installer, you can Modify your installation options after an successful installation of the UGX Launcher.

 

Options

 

You can modify currently the following options

  • Desktop Shortcut
  • Start Menu Entry
  • Autostart
  • Reset UGX Launcher settings

Depending on your selection, you can add, remove or keep the option.

  • Green: add this option
  • White: keep the option as it is
  • Red: remove this option
