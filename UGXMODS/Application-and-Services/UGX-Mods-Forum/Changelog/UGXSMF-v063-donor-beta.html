Summary

This release is supposed to be the last release before Open Beta.

The new landing page has been finalized, some usability improvements have been made and the experience for mobile users has been enhanced.

Important highlights from this release

  • New landingpage
    • Improved style and fully responsive now
    • Introduction of UGX-Mods, the team and it's projects
  • Usability improvements
    • Notification permission request will happen now once the user clicks on alerts windows rather than annoy the user on each visit.
    • Password length check and other usability improvements from registration and such are available on all pages now
    • Better remind and activate account page. Especially activate account page is less confusing now.
  • Better mobile experience
    • Navigation can be opened with a swipe to the left and closed with a swipe to the rightÂ 
    • optimized all images, reducing the total size of all images on the forum to 50% of the original size!
    • Introduction to service workers to turn our websites slowly into progressive web app
      • We do cache certain assets now, in order to allow a better offline experience
      • More work will be put into this in the future to further enhance your mobile experience on our sites
    • Page transitions will be visualized with a loading spinner / throbber from now on
  • Cookie banner display (yeah we are a bit late here) makes it finally back on the site, fully reworked and styled to work properly on mobile devices as well now
    • Notice: if you don't allow cookies, the banner will be always shown on each page load due to no technical limitations atm. (Ironically we need to store a cooke atm to save the sate, which the user just denied...)
  • Minor features
    • added rss icon + link to the feed for articles
    • better loading order and speed of assets - more will follow in the future
    • Updated navigation
    • Recent messages and unread messages received an style uplift as well.
  • CKEditor
    • a crash will be logged so that we can further investigate the issue and fix it
    • in case of a crash, we fallback to the old legacy text editor (which was restyled to fit in)
    • Updated to the latest version, including latest security fix
  • Bugfixes
    • the forum stats received a visual lift-up
    • guests can view public posts on the user profile now
    • smashed some logic bugs and errors
  • Development
    • Updated all dev dependencies and also updated client scripts to latest version range
    • better deployment of projects
