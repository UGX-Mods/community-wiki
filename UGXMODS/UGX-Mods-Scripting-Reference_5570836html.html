The UGX Scripting Reference ( www.ugx-mods.com/script ) is a website which lists all available Call of Duty: World at War Script Functions.

Addtionally, the site offers many neat useability features, like a recent- & favorite list, nice syntax and script examples.

UGX also added ultiity functions to the list, to offer newcomers hints to already solved scripting issues.

 

In attempt to have the most up-2-date Scripting Reference in the web - soon - UGX will allow community members to add / edit scripting functions.
