This tutorial will outline the needed steps to get Treyarch's XMODEL/XANIM Export tools setup in Maya. This will allow you to export your own 3D assets and animations to WaW.

Installation

To start off, you need to set up the

usersetup.mel

and

maya.env

files so Maya can find our tools.

The maya.env file creates the environment variables to point to the locations of plugins and scripts located in the tree, and should be put in

My Documents\maya\*VERSION*

(*VERSION* = 7.0, 8.0 or 8.5)
Example paths are:

MAYA_SCRIPT_PATH = C:\Program Files (x86)\Activision\Call of Duty - World at War\bin\maya
MAYA_PLUG_IN_PATH = C:\Program Files (x86)\Activision\Call of Duty - World at War\bin\maya\Maya8_Plugins

If you use Steam, example paths would be:

MAYA_SCRIPT_PATH = C:\Program Files (x86)\Steam\steamapps\cod waw\bin\maya
MAYA_PLUG_IN_PATH = C:\Program Files (x86)\Steam\steamapps\cod waw\bin\maya\Maya8_Plugins

These will tell Maya to look here for our plugins and scripts.
Don't forget to unzip the appropriate MLL scripts, either

Maya7_Plugins.zip

or

Maya8_Plugins.zip

The usersetup.mel file will tell Maya what to add on startup of Maya. This invokes the startup of our 'COD Tools' menu. Create a new plaintext document called "usersetup.mel" at the path specified below.

My Documents\maya\''VERSION''\scripts\usersetup.mel

Copy the following code into the file and save it.

source CODToolsMenu;
CODStartup;
CODToolsMenu;
