BBCode (also known as "Bulletin Board Code") is a markup language used on the UGX Forum and a lot of other forums you might use. It's the language that makes you able to easily add images, links or any other kind of formatting to your post without having any knowledge of the actual language that get's created: HTML.

We noticed some members are struggling with BBCode (i.e. not loading images, links not fully working, etc.), so we decided to explain what it is and how it's used on UGX.

BBCode on UGX

How does it work?

BBCode is really useful for creating/editing text areas with proper formatting without going through all the hassle of creating the markup yourself in HTML.

for example:

[b]this is some text[/b] will eventually result in this is some text

while

<span style="font-weight: bold;">this is some text</span> will also result in this is some text

We could all agree that the first way is much faster and more simple then the second one. The first one is using BBCode! We will explain the different tags down below.

BBCode Index:

Images:

Example

[img]{http://example.com/image.png/}[/img]
[img alt={Image Title} height={px} width={px}]{http://example.com/image.png/}[/img]

Bold:

Example

[b]{Text Here}[/b]

Italic:

Example

[i]{Text Here}[/i]

Underline:

Example

[u]{Text Here}[/u]

Strikethrough:

Example

[s]{Text Here}[/s]

Headings:

Example

[h{3-6}]{Text}[/h{3-6}]

Font Size & Font Color:

Example

[color={Hex Code}]{Text Here}[/color]
[size={px}]{Text Here}[/size]

Lists:

Example

[list]
[li]{Text}[/li]
[li]{Text}[/li]
[li]{Text}[/li]
[/list]

Text Alignment:

Example

[left]{Text Here}[/size][/left]
[center]{Text Here}[/size][/center]
[right]{Text Here}[/size][/right]

Quote:

Example

[q]{Text}[/q]

Post Quote:

Example

[quote author={User Name} link={http} date={timestamp}]{Text Here}[/quote]

Links:

Example

[url=http://google.com]Google[/url]

Code:

Example

[code={language code}]print("Hello");[/code]

Progress:

Example

[progress={Value in %}]{Title}[/progress]

Spoiler:

Example

[spoiler={Title}]{Text}[/spoiler]

Download Button:

Example

[download]{http}[/download]
