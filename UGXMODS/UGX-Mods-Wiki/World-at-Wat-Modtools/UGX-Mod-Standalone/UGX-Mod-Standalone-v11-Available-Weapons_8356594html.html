On this page you'll find all the weapons which are featured in the UGX Mod Standalone v1.1 together with their corresponding Filenames, Types and Elemental version.

Script Name: Name that can be used in script (.GSC) to give a certain weapon to a player.

Real Name: Name that will be shown on the player's HUD. The displayName key in weaponFiles is IGNORED by UGX Mod v1.1.

Type: Type of the weapon. Only certain types are available. (pistol / smg / rifle / dw pistol / dw smg / dw shotgun / sniper / lmg / shotgun / special / grenade / item).

PaP Version?: Tells us if the weapon has an upgraded version available. If it says 'No' you won't be able to Pack-a-Punch that gun without making a new version yourself.

Element?: Tells us if the weapon has an elemental version available. If it says '-' you won't be able to upgrade your gun in the Elemental Machine.
