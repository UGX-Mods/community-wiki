Summary

I'm happy to announce that first UGX MapPacker internal release which is built with Qt.
This version is a preview how the UGXMAPP will work.

Because UGXPLAY is not ready yet, an extra developer mode was introduced, which creates SQL queries to easily add the map / mod to UGX Play (and therefore to UGX Launcher )

Important highlights from this release

  1. First Qt Version
  2. New design
  3. SQL Developer Mode
