This page will describe the process how to submit your user generated content to UGX Play , which will be then available on the UGX Launcher as well.

 

Requirements

  • You need to agree to the Submit Policy for User Generated Content.
  • You need a preview image
  • You need to use the UGX MapPacker to create the archive of the content.

 

Naming your mod

We recommend to name your mod only with 0-9 A-Z a-z - _ to ensure it works for every system.

We highly recommend to exclude any version number from your mod! (old maps have them included because of legacy reasons)

Don't worry the version information can be viewed much butter within the UGX Launcher or on UGX Play !

What is version and revision

The revision is an increment able number, which starts at one and with every version / update it will be incremented by one.

The version is the text which will be displayed instead of the revision.

For example:

Revision Version Filename (since v0.8) Description
1 v1.0 <modname>.r1.7z 1st version of map
2 v1.2 <modname>.r2.7z 2nd version of the UGC
3 v1.2.3 <modname>.r3.7z 3rd version of the UGC
