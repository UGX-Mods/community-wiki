The UGX-Mods Forum supports a b8 spam filter since UGXSMF v0.8.8-beta .

Quick Guide

  • Every message is classified as unlearned.
  • A message should be learned as SPAM or HARMLESS depending on what type it is.
    • Only content which is identified as SPAM should be learned as such, this includes for example
      • Link to disallowed content (NSFW content, products, suspicious content, etc.)
      • Repeatedly the same content by different accounts
      • Whatever you think is SPAM
    • Content which doesn't follow our rules, but is written by an individual and not a SPAM Bot is therefore no spam and is harmless, but shouldn't be allowed (hide or remove it)
  • Messages can be classified. This returns a score, based on the previous learned words.
  • So the more content has been learned, the more likely and accurate is the score.
  • The score goes from 0.0 (no spam) to 1.0 (spam)
  • At the moment we consider SPAM which is >= 0.7
  • Visit https://www.ugx-mods.com/forum/moderate/area=antispam-posts for b8 overview
  • Be sure to learn from time to time some messages. (not all messages of the forum need to be learned, but different written types of SPAM and HARMLESS content is important)
  • Messages can be reclassified (over time, so their score can change)
  • Messages can also be unlearned again, previously classified content won't update though!

Anti Spam Forum Posts Overview

The page at https://www.ugx-mods.com/forum/moderate/area=antispam-posts shows all forum posts (recent first) and the b8 status.

You can use the filters to reduce the amount of posts.

Display / Status

The current b8 status is displayed on the forum post with a robot icon.

If no robot is present, b8 has not been executed on it.

Otherwise a robot icon, followed by the score is shown if it has been classified.

An additional Icon will be shown if it has been learned.


Example

Red - dead robot. This content is SPAM (>0.9). The stop / attention icon means that this content has been learned as SPAM.

Red - angry robot. This content is most likely SPAM (>0.7 && <0.9)

yellow - confused robot. This content is eventually SPAM (>=0.35 && <0.7) | This will be most likely shown for many posts during the learning phase.

Green - happy robot. Content is okay (<0.35). The okay hand means this content has been learned as HARMLESS.

You can hover over the icons or score to get an tooltip when this content was classified / learned last. (updated means, that it takes either classified at or learned at depending on which is more recent)

Forum Message Overview
