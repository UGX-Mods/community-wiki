Summary

This release contains some minor new features, a couple of improvements and some bug fixes.

Important highlights from this release

  • Search results can be ordered now
  • Added topic preview. Hover over any topic on the message index and click on the preview icon to get a small preview of the topic.
  • Answered questions in any Help Desk can be viewed directly by clicking "SOLVED"
  • Added topic specific searching. Now you can easily search within a topic.
  • Fixed pagination for items with more than 250 pages
  • Improved page styles (SendTopic, Poll, Search, mobile view tweaks)
  • Staff changes:
    • Improved member list with avatar support
    • Improved mailcheck (identify users with broken emails)
    • Improved page style of more moderator / admin pages
