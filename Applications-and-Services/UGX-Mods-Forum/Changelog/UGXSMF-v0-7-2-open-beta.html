Summary

Last minor release before going live! ¬†Added emoji support almost everywhere. We also added new emotes and exclusive emotes (donors only) to our reworked emote picker.

Important highlights from this release

  • Added emoji support
    • You can start typing : for autocompletion
    • Optimized autocomplete to handle large suggestion lists
    • Native written emojis will be converted properly into the colon format / into an image so everyone can see them properly
    • Emoji style can be changed in your Forum Settings under Look & Layout .
  • Improved emote picker and added exclusive emotes to certain groups, especially for donors
    • Added new emotes and categories
  • Added newsletter subscriptions settings to the UGX-Mods profile
    • manage your subscriptions and review settings within your UGX-Mods profile
  • General Improvements
    • Updated external libraries to their latest stable version (mostly bug fixes and performance improvements).
    • Removed code of the originally planned usermenu and UGX Chat v2 from 2013 (topic: https://www.ugx-mods.com/forum/index.php?topic=1315 )¬† üò•
    • The privacy / GDPR tools have been improved and extended (more data to export from core systems and plugins)
      • some external plugins won't store personal data after deletion anymore
    • Many tiny improvements to enhance the quality of the forum
    • We further refined our privacy documents and will make them public once we move to production
  • System
    • Added Sentry ( https://sentry.io) ¬†to monitor errors in the front- & backend. This will improve quality over time and let us react even faster to issues.
    • Newsletter system (powered by phplist) has been implemented and customized to fit our new blue design.
    • Connected phplist with the UGX-Mods forum to sync information between both instances.
  • Bugfixes
    • Fixed errors found in error log
    • Fixed lots of tiny bugs and style issues
