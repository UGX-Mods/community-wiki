Summary

Thanks to all for your great feedback . This release contains a couple of improvements and (critical) bug fixes.

Important highlights from this release

  • Nicer beta footer (scroll to bottom) & nicer changelog on site

  • Fix new message bubble (yeah we noticed it too! )

  • Refactor of BetaAccess feature for Work in Progress maps:
    • Works in all WIP Topics now
    • Added success, info and error banners
    • Info / Status about your beta state inside the topic
  • sharper icons & avatars in chrome & IE
  • Fixed critical bug with moderation (Un-)Hide feature.
    • Normal users wouldn't see posts because pageindex (total page count) would be calculated wrong
    • Improvements for Moderators
  • Bug fixes (errors you didn't even noticed )
  • Not relevant for you, but still nice to know
    • Upgrade of SEO Module
    • Deployment Script to make updates easier
