Summary

Donor release of the new UGX-Mods Forum with many new improvements and features. It has been almost a year now since the last update and a lot has happened since then!

Important highlights from this release

  • New registration process (not visible / activated yet)

  • Preparations for Social Login - soon you can authenticate / link some 3rd party accounts like Twitter, Facebook, Steam, ...

  • Mail check for admins and registration mail status for fresh registered users

  • CKEditor major improvements, features and fixes

    • Added autocomplete for emotes, users and keywords

      • Type in @username to create a link to an user

      • Type in #UGX to create a link to our homepage

      • Type in :nyan: to see autocomplete of emotes in action¬† üôÇ

    • Added Gallery and Slider, now you can embed images like a pro! (in the non-beta version it will simply show all images as an fallback)

    • Added auto linking, type in www.ugx-mods.com and it magically turns into a link üëç

    • Added some new buttons which were missing

    • Option to switch to small, extended and advanced editor.

    • Better media embed option, with caching support!

      • you need to manually add it now , by clicking the "Insert Media Embed" Button

    • Insert emotes has finally a recent used list

  • ReCaptcha instead of answer + question / annoying unreadable images

  • Styling of all major pages - enjoy nicer pages

  • Many improvements to mobile version

    • including improved Navigation & Dropdowns (go check it out on your smart device¬† üôÇ )

  • timezone instead of timeoffset - the time is always your local time now, even during DST (daylight saving time)¬† üëç

  • Added support for Gravatar ( https://gravatar.com ) - just select it in your settings (be aware that it won't be shown in the non-beta version yet!)

  • Added search below navigation + ability to go to advanced search

  • Fix topic sorting¬† üòÅ yay

  • New forum boards child display

  • New login popup, simply click the login button
  • Share & Subscribe Widgets for forum posts and news
  • New shiny improved usercard - it works on all avatars now!
