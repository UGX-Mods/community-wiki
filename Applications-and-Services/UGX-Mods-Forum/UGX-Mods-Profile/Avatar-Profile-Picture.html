Please note

We will remove any avatar in the future if it doesn't met the requirements.

In an upcoming update, remote URL's will be deprecated and only Gravatar or direct upload will be supported.

But don't worry, we will notify you about these changes and also use your existing remote URL avatar and upload it to our servers (if valid, etc)

Requirements

Rule Min Max
Width 64 512
Height 64 512
File size 0 256kb (gif can be up to 2mb!)
File type PNG, GIF, JPG, JPEG (JPEG is preferred)

Upload [RECOMMENED]

You can upload your own avatar and modify it with our Avatar Editor .

Gravatar

Another fast option is https://gravatar.com/ because it's easy to setup and many sites do support it.
Simply register with your email which you are using on our forum, set your avatar on Gravatar and select the gravatar option on your UGX-Mods Profile.

External URL

If you prefer to use an URL, we recommend https://imgur.com/ . Advanced users can also use their own webspace (or of course other providers).

Please note that the URL must end with one of the allowed filetypes!

Your URL should start with https:// (encrypted TLS/SSL)


You can also use the Avatar Editor to fetch the image and modify it to meet the requirements above.
