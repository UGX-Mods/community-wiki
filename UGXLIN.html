What is it?

The UGXL Installer is an application which will install the UGX Launcher and register the application in windows.

So, to get the UGX Launcher , you need this application.

What does it do?

It will download the latest version and then extract it either to the default Installation path, or your self defined, custom path.

It will install all necessary files to start the  UGX Launcher .

It will also

  • register the application to the Programs & Features of Windows.
  • register the ugxl:// URL scheme. This is needed so you can open the UGX Launcher through your web browser for example.
  • (optional) register to the autostart list, so once you boot up your system, the UGX Launcher will start as well.
  • (optional) a shortcut to execute the UGX Launcher
  • Runtime Check
  • Security Applications check
  • Firewall Whitelisting
  • Online Check
