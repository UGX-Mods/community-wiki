This guide will explain how to translate an application into your language.

It will also explain how to submit it to be part of the next update and how to maintain it when there is a new update with new translatable text.

How to translate

You need Qt Linguist to create / open the *.ts files

https://doc.qt.io/qt-5/linguist-translators.html

How submit

Once you are done with translating, you need to submit it.

  1. You need a valid UGX-Mods Help Desk Account (more info here!) and UGX-Mods Forum Account.
  2. Raise a new ticket
    1. Summary: Localization of <Application> into <Language> ( where Application is UGX Launcher oder UGX Installer and Languge is the one you selected in step 5 of How to translate )
    2. Description: Your description, if there are any issues write them down there as well
    3. Attachment: Upload the *.po file there
  3. Keep an eye on this ticket and wait for instructions
