The map view will display all maps with a preview image.

Map View Top bar

The top bar contains the current navigation path (also known as a "Breadcrumb" ).

The back / forward buttons do act like the ones in a web browser.
A right click opens the context menu display the history:

The blue highlighted item is the current page.

 

The right side contains Map filter options and Map View Sorting .

These allow you to find maps which meet certain criteria you specify.

You can find maps which were released during a certain year, or maps which have specific features you like (or don't have features you dislike!)

Map View List

Each map is displayed as a rectangle with the map name, display image, and the installed state (in a color and via text description) of the map, as well as three buttons to interact with.
