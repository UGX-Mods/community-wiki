The old UGX Map Manager had issues with Anti-Virus and Firewall applications blocking its access to our download server.

With the UGX Launcher we are trying our best to solve this problem by detecting your installed programs and adding rule exceptions automatically. If we can't add an automatic rule or don't recognize your anti-virus/firewall, we will do our best to link you to detailed instructions on how to fix it yourself or report the issue to us.

 

(This is currently only enabled in the UGXL Installer so that download issues are discovered and potentially solved before the user tries to use the UGX Launcher)
