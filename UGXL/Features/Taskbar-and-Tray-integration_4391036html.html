UGX Launcher will add an taskbar icon which offers some quick information.

Also the taskbar element will have options and display the overall download progress.

 

Taskbar Jump List

The UGX Launcher supports the Windows "Jump List" menu (right-click on the taskbar icon from the start bar). This menu has quick links to various UGX Launcher menus/views/tasks.

 

Taskbar Progress

UGX Launcher will display the download & install progress of active downloads on the taskbar.
