UGX Launcher has full support for installing and using T4M .

 

UGX Launcher will check if your installed version of CoDWaW supports T4M and check if T4M is installed already and up-2-date.

The user will be asked to automatically install T4M once they try to download a map which requires it.

(map which requires T4M)

 

Filter for T4M

UGX Launcher offers a T4M filter to include / exclude T4M featured maps, for more information check Map filter options. If you wish to hide all maps which require T4M, you can do so easily with the Filters menu.
