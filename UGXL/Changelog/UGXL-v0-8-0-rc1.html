Summary

I'm happy to announce the first release candidate (RC1) towards donor release.

Major new additions have been implemented, the design is almost finish now!

Important highlights from this release

  1. Reworked (blue) design
    1. Treelist has animated collapse / expand
    2. Treelist is resizeable now
    3. Style files splitted (support fur multi loading stye files)
    4. New UGXL Logo by Andy King
  2. New GameManager with current playing + mod.
    1. Recognize game already running (once ugxl is started)
    2. identify World at War Version
  3. Full steam support (start WaW over steam)
  4. New Settings Manager
    1. new download speed control
    2. more settings
  5. Resume active downloads on app start
  6. Windows Task Bar addtions: Overlay Icon, Jump List and Progress bar
  7. T4M support
    1. identify if game version can support T4M
    2. T4M download / update support
  8. Uninstall support
    1. use paramater "-uninstall" to open an uninstall window through UGXL. (shortcut in startmenu and MSW Program & Features support)
  9. Logout support
  10. Warning popup for wrong waw path and low hdd space
  11. Save recent played maps
  12. Updated to Qt 5.6
  13. Max download speed is shared / divided between all active downloads. (1Mb/s for 3 downloads is now 333kb/s each, instead of 1Mb/s which would make a total of 3 Mb/s)
  14. Minor bug fixes and improvements

Known issues

Download Manager is buggy, report all issues

Updater of rev 21 / 20 is broken (7z extration failed. Alexander Diller is investigating it)

Be carefull, the updater is broken (you'll see) but just hit the button and it should work.

You then will get a 7z extraction error

 

close the application and locate the installation root of UGXL.

Remove all files and go to %TEMP%\UGX Launcher or (C:\Users\%username%\AppData\Local\Temp\UGX Launcher)

and extract the full.r22.7z by yourself into the install dir

 

Now start UGX Launcher.exe =)
