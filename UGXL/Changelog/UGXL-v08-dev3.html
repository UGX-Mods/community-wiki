Summary

I'm happy to announce that the next developer version (#3) is available.
Designer requested features has been implemented. First release guided with UGX Versioning.

The updater should work without any issues!

Important highlights from this release

  1. Auto-reload theme for all elements now fully working.
  2. Fixed high CPU usage again (from ~10 % to ~0)
  3. *.bik movie handling (add & remove of bik, can be tested with Oil Rig)
  4. Window position and size is saved now
  5. Added settings button to login and updater (to access at a minimum the network settings)
