The UGX Launcher tells me there are disallowed files

The UGX Launcher is verifying each map folder if it's up-2-date and if there are any files which shouldn't be there.

 

What files are not allowed?

Please note

The files are disallowed because they are in the map folder. If you move them out, everything will be fine.

The UGX Launcher will only offer to delete them. So if you want to move them out, click NO or  Skip and  move them out of the map folder

file (type) description
*.iwd

Only *.iwd files which were uploaded by the mapper are allowed!

Any modifications or additional *.iwd files are disallowed!

*.ff

Only *.ff files which were uploaded by the mapper are allowed!

Any modifications or additional *.ff files are disallowed!
mod.arena Is allowed / whitelisted.
version.txt legacy file of the UGX Map Manager (will be removed silenty once UGX Launcher is stable)
missingassets.csv Auto Generated file by WaW. Will be ignored / is allowed.
*.cfg CFG files are currently allowed.

Any (sub) folder within the mod is disallowed as well!

 

Why are those files or file modification disallowed?

Usually, a modification means that a player tried to exploit a script to make the game easier. - We try to prevent this.

Additional *.iwd and *.ff could be loaded by the game automatically, which could invoke new scripts or mess with the map / mod. - We try to prevent this.

 

What are the consequences if I don't delete them?

The map will be marked as Corrupt and but you still can play the map.
