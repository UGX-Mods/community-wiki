The UGX Launcher has different update modes.

 

The default and for most users recommended one is the stable update mode.

 

Under Settings > Account it is possible to change the update mode,
restrictions apply depending on your UGX-Mods Forum Group and other criteria.

 

List of Update Modes

Name Requirements Information
stable none Default Mode. This update mode ensures that the UGX Launcher only updates to tested and verified versions.
beta For UGX-Mods Staff and UGX-Mods Donators Beta Mode. Be the first to get the latest Features and Improvements of the UGX Launcher!
To participate, you need the UGX-Mods Donator Group Stats, which you can obtain by making a donation .
experimental

For UGX-Mods Staff and UGX-Mods Donators,

who applied for experimental.

Experimental Mode. Help by testing the latest development builds of the UGX Launcher.

To participate, you need the UGX-Mods Donor Group Status and need to apply: "coming soon"

dev Only for UGX-Mods Developers This mode is only for UGX-Mods Developers, to test new features before we access it to experimental, beta or stable.
