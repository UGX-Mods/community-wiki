Find the map or mod you want to install using one of the below methods.

Use the

  • Search
  • Map filter options
  • Map View Sorting

to quickly find the map you desire.

 

You can either left click on "Install" in the Map View or right click on the map in the Map View or Tree List and choose "Install" to install it.

Then the download will start.
