The Download Manager will help you to track downloads, prioritize them and get useful information about each download

Access the Download Manager

In order to access the Download Manager , you need to have a download started.

Then simply click the download bar to open the Download Manager .

 

How to use the Download Manager

 

Each map is represented as such:

and displays various download information.

 

Change download priority/order:

If there are enough downloads in the queue, it is possible to reorder them.

The double arrow up will force-start the download and will move an active download to the queue (replace download)

The double arrow down can be used for active downloads, to move them to the end of the download queue and stop it.

The normal arrow up and down can be used to change the order of the Queue.
