The UGX Launcher will require Authentication with your www.ugx-mods.com (forum) account.

We decided this for multiple reasons:

  • Prevent download spam.
  • More features for the user
  • Better control & moderating
  • Access control based on the user

The authentication over the internet will be transmitted over an encrypted connection. [1]

Everyone can access the UGX Launcher, as long as he isn't banned / blocked from www.ugx-mods.com
