Installing the Patches

This is a guide on how to patch a non-Steam edition of World at War to v1.7. This must be completed BEFORE installing Modtools or attempting to play custom maps.

Install each patch in the order that they are listed below, one at a time. Ensure to only install v1.4.1-DEU if you have a german copy of CoD WaW.

If you encounter any problems during the installation of any patch, feel free to ask for help in our Help Section of the forum.

What is a checksum?

A checksum is a sort of 'fingerprint' used to verify the integrity of a file after it has been downloaded. To ensure that the file you've downloaded is identical to the one on our server you can compare our checksum hashes with ones you calculate on your own using one of the hash functions we've listed.

How to validate the exe?

Download a tool such as HashCheck if you want a hashing tool which is integrated into windows explorer for easy access to verify checksums. With HashCheck installed right click the downloaded file, go to properties, and navigate to the Checksums tab. Once there you can compare the hashes to the ones provided on this page to ensure you have the same file provided on our website. If the hashes are different DO NOT RUN THE FILE . You may attempt to retry the download and verify again but executing the file could have unwanted effects such as an improper installation or even malicious code executing on your machine.
