In the current version of the BO3 Modtools (Beta), Treyarch also did not include box location prefabs with the tools at this time, so you would have to grab stuff from The Giant mapfile. 

Box Location Prefabs

box_start.map

box_01.map

box_02.map

box_03.map

You can make as many more box locations as you want by just changing the KVPs on the prefabs.

Placing the Prefabs

From the Entity Browser, drag a Misc\_prefab into your scene and navigate to the prefab in the file browser window.
